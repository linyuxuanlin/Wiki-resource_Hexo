#### 通过 GitHub 下载的方法

（熟悉 GitHub 请忽略）

* 电脑端   
可使用 `Find file` 搜索
点击 `Download` 或 `View Raw` 即可下载

* 手机端   
`Jump to file` 搜索你想要的 app
右下角 `Desktop version` 切换至电脑端界面
用 `Download` 或 `View Raw` 下载
